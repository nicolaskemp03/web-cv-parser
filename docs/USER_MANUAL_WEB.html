Manual de Usuario - Konexa CV

Bienvenido a la herramienta automatizada de procesamiento de Currículums de Konexa. Esta aplicación utiliza Inteligencia Artificial para extraer, ordenar y dar formato a los currículums de forma automática.

1. Ingreso al Sistema

  • Dirígete a la URL corporativa asignada.
  • Ingresa utilizando la contraseña maestra proveída por la administración.

2. Importación de Candidatos

Opción A: Importar de Teamtailor Recomendado

Esta es la manera más rápida si el candidato ya está en el ATS.

  • Haz clic en Importar de Teamtailor en el Dashboard.
  • Busca por nombre, correo o ID numérico. (Nota: La búsqueda por nombre consulta a los últimos 210 postulantes recientes).
  • Presiona Extraer CV. El PDF original se descargará y leerá automáticamente.

Opción B: Subir Manualmente

Ideal para CVs recibidos por correos externos o WhatsApp.

  • Haz clic en Subir CV.
  • Arrastra tu archivo .pdf o .docx.
  • Selecciona el idioma y presiona Procesar Documento.

3. Edición Interactiva

En el Dashboard, presiona el ícono de lápiz en cualquier candidato para abrir el editor:

  • Datos Básicos: Corrige información de contacto.
  • Acerca de: Utiliza el editor de texto enriquecido (Negritas, Cursivas).
  • Experiencia y Educación: Reordena las experiencias utilizando la herramienta de "arrastrar y soltar" (Drag & Drop) en la esquina superior de cada tarjeta.
  • Stack: Escribe tecnologías nuevas y presiona Enter para añadirlas, o bórralas con la "X".

4. Generación del Formato Konexa

Una vez satisfecho con la edición, haz clic en el ícono de descarga (PDF). El sistema ensamblará un documento PDF premium utilizando la plantilla oficial de Konexa (modo oscuro, diseño estandarizado), dejándolo listo para ser enviado al cliente.
