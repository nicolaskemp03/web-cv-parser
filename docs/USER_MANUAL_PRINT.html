Konexa CV Parser

Manual de Usuario Oficial

La herramienta de procesamiento de CVs de Konexa permite la estandarización automatizada de currículums empleando Inteligencia Artificial avanzada, transformando documentos en crudo en un formato PDF premium, homogéneo y listo para clientes.

1. Acceso al Sistema

Para ingresar al sistema, diríjase a la dirección web interna proporcionada por el administrador e ingrese con la contraseña maestra autorizada del equipo de reclutamiento.

2. Flujo de Importación de Candidatos

Existen dos métodos para ingresar candidatos al sistema:

A. Integración con Teamtailor (Recomendado)

  • En el panel principal, presione el botón Importar de Teamtailor.
  • Utilice el buscador para encontrar al candidato mediante su Nombre, Correo Electrónico o ID.
  • Una vez localizado, presione Extraer CV.
Nota: La búsqueda por nombre en Teamtailor es un proceso optimizado que busca de forma inteligente dentro de los ~210 registros más recientes. Para candidatos antiguos, se recomienda usar el correo electrónico.

B. Carga Manual de Documentos

  • En el panel principal, seleccione Subir CV.
  • Seleccione el documento (Soportado: PDF o DOCX).
  • Indique el idioma original del documento para maximizar la precisión de la lectura por IA.
  • Presione Procesar Documento.

3. Proceso de Edición (Human-in-the-loop)

El sistema extraerá más del 90% de la información automáticamente, pero permite supervisión humana total. Presione el ícono de edición (lápiz) sobre cualquier candidato para abrir el editor:

  • Datos Personales: Campos estándar de texto para ajustar ubicación y contacto.
  • Acerca de: Editor de texto enriquecido. Puede seleccionar partes del texto y aplicar formatos básicos (Negrita, Cursiva).
  • Organización de Experiencia/Educación: Haga clic sostenido sobre la esquina superior izquierda de cualquier tarjeta de experiencia y arrástrela (Drag & Drop) para alterar el orden cronológico o de importancia según requiera el cliente.
  • Stack Tecnológico: Ingrese nuevas herramientas escribiéndolas y presionando la tecla Enter. Elimine elementos irrelevantes haciendo clic en el ícono de cierre.

4. Generación de Exportables (PDF)

Cuando los datos del candidato estén pulidos y revisados:

  • Desde la vista de edición o desde el panel principal, presione el botón Generar PDF (ícono de descarga).
  • El sistema compilará la información utilizando la plantilla oficial corporativa de Konexa (Dark Mode).
  • El navegador iniciará la descarga automáticamente del documento PDF resultante.
